What is Erithacus pardus?

It is a project for the class of Artificial Intelligence, the main purpose is to analyze the affect, mood or emotion in a tweet. The project is currently being developed in order to have more data and more ways to provide more information via a single tweet.

Why?

Because we love Artificial intelligence. One of the goals is to apply the knowledge into a project that can help or be helpful in any area. Therefore, this project does not have a commercial purpose.

Time

The time period of development is not only for the course. The main idea is to keep developing this project. We are looking for more ideas and more interesting things.

Main point

It is focus on determine the affect and also to provide more information about the person who is witting the tweet. The main point in this project is to detect if a person has suicide tendencies, terrorism or any other destructive behavior.
